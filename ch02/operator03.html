연산자

6. 조건 선택 연산자(삼항 연산자) : (condition ? true : false)
- 조건에 대해서 true 일 때와 false 일 때 다르게 실행하는 연산자
- 간단한 if-esle문의 대용으로 사용

연산자의 우선 순위

1. 단항연산자(++, --, !)
2. 산술연산자 (*, /, %, +, -)
3. 비교연산자(>, >=, <, <=, ==, !=, ===, !==)
4. 논리연산자(&&, ||)
5. 삼항연산자(condition ? true : false)
6. 대입연산자(=, +=, -+, *=, /= )
