연산자(operator)

1. 산술연산자 : (+, -, *, /, %)
- / : 나누어서 몫을 구함, % : 나누어서 나머지를 구함
2. 증감연산자 : (++, --)
- 증가연산자 : 변수의 앞 또는 뒤에 붙여 사용, 변수의 앞에 있으면 전위증가연산자 / 뒤에 있으면 후위증가연산자
- 감소연산자 : 변수의 앞 또는 뒤에 붙여 사용, 변수의 앞에 있으면 전위감소연산자 / 뒤에 있으면 후위감소연산자
3. 비교연산자(관계연산자) : (>, >=, <, <=, ==, !=, ===, !==) 결과가 논리값 형태로 표현, true 또는 false
- >, >=, <, <=, ==, != : 값만 비교하여 true 또는 false
- ===, !== : 값과 타입을 한꺼번에 비교하여 true 또는 false
