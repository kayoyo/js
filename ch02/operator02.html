연산자(operator)

3. 비교 연산자 : >, >=, <, <=, ==(equal), !=(not equal), ===, !==
- ==, != : 값만 비교해서 true 또는 false를 출력
- ===, !== : 값과 데이터 타입을 모두 비교해서 true 또는 false를 비교
- 우선순위 : 첫번째 >, >=, <, <= 두번째 ==, != 세번째 ===, !==
4. 논리 연산자 : &&(논리곱 AND) || (논리합 OR) !(논리 부정 NOT)
- 논리연산자의 결과는 논리값(true or false)
- 비교연산자 > 논리연산자
- 우선순위 : ! > && > ||
5. 대입연산자 : =
- 복합대입연산자(할당연산자 assignment) : +=, -=, *=, /=, %=
